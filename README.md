# Lumière Restaurant

A restaurant landing page built with React and Vite.
It demonstrates a complete front-end setup with navigation, menu sections, gallery, contact information, and WhatsApp reservation integration.

🔗 **[Live Demo](https://murilotecoteco.github.io/Restaurante-Vite-React-/)**

---

## Overview

The website includes:

- Responsive layout for desktop, tablet, and mobile
- Sticky navigation bar with smooth scrolling
- Interactive menu section
- Image gallery
- Contact section with location and working hours
- WhatsApp reservation button
- Mobile-friendly navigation menu
- Animations and transitions

## Installation

Clone the repository:

```bash
git clone https://github.com/murilotecoteco/Restaurante-Vite-React-.git
```

Navigate to the project folder:

```bash
cd Restaurante-Vite-React-
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## Production Build

Generate production build:

```bash
npm run build
```

Preview the build locally:

```bash
npm run preview
```
